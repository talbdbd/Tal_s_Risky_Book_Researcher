# Tal_s_Risky_Book_Researcher

## Description

The goal of this project was to refactor a RESTful API Book Search into a GraphQL API.

- My motivation was to refactor the book search successfully.
- I built this project to further my skills and proficiency in GraphQL API as well as MERN stack.
- The main solution I strove for was to make the book search run on GraphQL API given the starter code.
- My main learning experience from this project was how to effectively refactor code from RESTful to GraphQL.
